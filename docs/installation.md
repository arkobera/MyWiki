# Installation

## Stable release

To install Python Boilerplate, run this command in your terminal:

```sh
uv add Python-Boilerplate
```

Or if you prefer to use `pip`:

```sh
pip install Python-Boilerplate
```

## From source

The source files for Python Boilerplate can be downloaded from the [Github repo](https://github.com/arkobera/Python-Boilerplate).

You can either clone the public repository:

```sh
git clone https://github.com/arkobera/Python-Boilerplate
```

Or download the [tarball](https://github.com/arkobera/Python-Boilerplate/tarball/main):

```sh
curl -OJL https://github.com/arkobera/Python-Boilerplate/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd Python-Boilerplate
uv sync
```
